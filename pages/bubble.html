Algorithm Info

Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

Best Time: O(N)
Avg Time: O(N²)
Worst Time: O(N²)
Space: O(1)

Algorithm Pseudocode

procedure bubbleSort(A : list)

  n = length(A)

  for i = 0 to n - 1 do

    for j = 0 to n - i - 1 do

      if A[j] > A[j + 1] then

        swap(A[j], A[j + 1])

  element at n - i - 1 is sorted
