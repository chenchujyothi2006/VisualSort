Algorithm Info

Selection Sort repeatedly selects the minimum element from the unsorted portion of the array and moves it to the beginning.

Best Time: O(N²)
Avg Time: O(N²)
Worst Time: O(N²)
Space: O(1)

Algorithm Pseudocode

procedure selectionSort(A : list)

  for i = 0 to length(A) - 1 do

    min_idx = i

    for j = i + 1 to length(A) do

      if A[j] < A[min_idx] then min_idx = j

    swap A[i] and A[min_idx]
