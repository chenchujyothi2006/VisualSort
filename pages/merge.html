Algorithm Pseudocode

procedure mergeSort(A, left, right)
  if left >= right then return
  mid = (left + right) / 2
  mergeSort(A, left, mid)
  mergeSort(A, mid + 1, right)
  merge(A, left, mid, right)

Merge Sort follows the Divide and Conquer technique. It recursively divides the array into two halves, sorts each half, and then merges them into a single sorted array.
