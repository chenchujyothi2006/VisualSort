Algorithm Info

Merge Sort divides the array in half, recursively sorts both halves, and then merges them back together.

Best Time: O(N log N)
Avg Time: O(N log N)
Worst Time: O(N log N)
Space: O(N)

Algorithm Pseudocode

procedure mergeSort(A, left, right)

  mid = (left + right) / 2

  mergeSort(left, mid)

  mergeSort(mid + 1, right)

  merge(A, left, mid, right)
