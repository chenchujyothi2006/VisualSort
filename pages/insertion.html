Algorithm Info

Insertion Sort builds the sorted array one item at a time by shifting larger elements to the right.

Best Time: O(N)
Avg Time: O(N²)
Worst Time: O(N²)
Space: O(1)

Algorithm Pseudocode

procedure insertionSort(A : list)

  for i = 1 to length(A) - 1 do

    key = A[i], j = i - 1

    while j >= 0 and A[j] > key do

      A[j + 1] = A[j]

      j = j - 1

    A[j + 1] = key
